## 使用教程

1. 在 `HoshioBot\hoshino\modules`目录下使用以下命令拉取本项目

   ```
   git clone https://github.com/pcrbot/determine
   ```

2. 将项目中的`determine_img`文件夹放在`res\img`目录下

3. 在`HoshinoBot\hoshino\config\__bot__`文件的MODULES_ON 加入 'determine'

4. 完成后重启Hoshino即可！
